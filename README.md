# 🍎 ひきざんゲーム

ロイロノートで使える、子供向けの引き算練習ゲームです。

## ✨ 特徴

- **3つのレベル**
  - 🌱 やさしい (1〜10)
  - ⭐ ふつう (1〜20)
  - 🔥 むずかしい (1〜99)
- **全10問**で1ゲーム
- やさしい・ふつうレベルでは**絵文字で視覚的に**引き算を表現
- 4択クイズ形式でタブレット操作に最適化
- 正解時の**紙吹雪エフェクト**
- かわいい**キャラクターからの応援メッセージ**
- 結果画面で**星評価**

## 🎮 使い方

### ブラウザで遊ぶ
`index.html` をブラウザで開くだけ

### ロイロノートで使う
1. `index.html` をロイロノートの「資料箱」または「カード」に追加
2. Webカードとして子どもたちに配布
3. それぞれのタブレットで遊べます

## 🛠 技術

- 単一HTMLファイル（外部依存なし）
- キャラクター画像はBase64で埋め込み済み
- フォント: Google Fonts (Kosugi Maru, Mochiy Pop One)
