# くらいどりタワー（3桁の数の位取りパズル 3D）

小学校2年生向けの算数3Dゲーム。上から落ちてくる「100の塊」「10の束」「1のバラ」を、
正しい位のタワー（百・十・一）に落として、ターゲットの3桁の数をピッタリ作る。

## 遊び方

- 平地に立った**一人称視点**。目の前に3本の巨大タワー、はるか上空（雲の上）からブロックが降ってくる
- ターゲットはタワーの奥にいるモンスターの頭上の吹き出し「**ほしいパワー: 〇〇〇**」に表示される
- ブロックは量が見た目でわかる形: 1=小さなキューブ、10=キューブが10個つながった棒（節目入り）、
  100=10の棒が10本合体した10×10グリッドのプレート
- 各タワーの手前には**光る円盤の台座**があり、立体の数字（100/10/1）がゆっくり浮遊している
- 降ってくるブロックを **←→キー** または **ドラッグ** で動かして、正しいタワーへ
  - 大きなプレート = 100 → 百のくらい（オレンジ）
  - 長い棒 = 10 → 十のくらい（緑）
  - 立方体 = 1 → 一のくらい（青）
- 間違ったタワーに入れると画面が赤く光るミスエフェクト
- **繰り上がり（ガッチャンコ）**: 1の立方体が10個たまると縦一列に合体して「10の棒」に変身し、
  弧を描いて十の位のタワーへ飛んでいく（十の位→百の位も同様）
- **「0」のひっかけ問題**: ターゲットに 305 や 420 など 0 の位が混ざる。
  0 の位のタワーにブロックを落とすと、タワーが激しく揺れて煙が出るミス演出
- 合計がターゲットとピッタリ一致すると、タワーのブロックが上空へパシャーンと弾け飛び、
  星パーティクル＋紙吹雪が舞い散る大爆発クリア演出 → 次の問題へ
- **モンスター育成（カンジモン流）**: タワーの奥にお腹をすかせたちびモンスターが浮いていて、
  頭上の吹き出しでターゲットの数をおねだりしている。クリアするとタワーから光の粒子が
  シュシュシュッと飛んでいき、モンスターがピョンピョン跳ねてクルクル回って大喜び
- **まんぷくゲージ＆進化**: 3問クリアでゲージ満タン →「レベルアップ！しんか！」が3D空間に
  ポップアップし、体の色が変わってパーツが増える（Lv2: 耳 → Lv3: 金のつの → Lv4: 王冠）。
  レベルは localStorage に保存されるので、次に遊ぶときも育てた姿のまま

## 起動

```bash
npm install
npm run dev
```

## 技術構成

- Vite + Vanilla JS + Three.js
- `src/main.js` — シーン・カメラ・ライト・操作・メインループ
- `src/towers.js` — 3本の透明タワーとレーン定義
- `src/blocks.js` — 落下ブロック（100/10/1）の生成
- `src/game.js` — 問題生成・落下・判定・積み上げ・クリア処理
- `src/effects.js` — ミス時の赤フラッシュ＆シェイク、紙吹雪パーティクル
- `src/ui.js` — ターゲット・現在値・メッセージの HTML オーバーレイ

## 現在の仕様（ベース版）

- 各位の数字は 0〜5（百の位は 1〜5）。十・一の位はそれぞれ 35% の確率で 0 になる（`game.js` の `newProblem` で調整可能）
- **オーバーフロー**: 合計がターゲットを超えるとモンスターがフラフラ気絶し、
  タワーのブロックが全部弾け飛んで同じ問題をやり直し（入れすぎは実際に起こる失敗）
- **ゴミ箱（ブラックホール）**: 画面の両はしに、渦巻きテクスチャが回転するネオン紫〜ピンク＋水色の
  ブラックホールがあり、ネオンリングと光の粒が回りながら「すてる ばしょ」として目立つ。
  ブロックを持っていくとクルクル回りながら吸い込まれて消滅（ノーペナルティ）。
  位置はタブレット幅を基準に画面端へ寄せ、タワーから十分離れた場所に配置される。
  いらないブロックを上手に捨てるとモンスターが「ナイススルー！」「えらい！」と吹き出しで褒めてくれる
- **リアルタイム表示**: モンスターの右側に「ほしいパワー」、左側に「いまの ごうけい」の
  大きな吹き出しを常時表示
- **出現確率の最適化（無理ゲー防止）**: もう足りている位・0の位・入れると超えるブロックは
  出現率が大幅に低下し、同じ不要ブロックの連発もほぼ出ない。
  ラッシュは序盤には出ず、慣れてきたころに短めに発生する。ラッシュの50%は
  「1の立方体×4連続のくりあがりミッション」になる
- **スピードの緩急**: 合計がターゲットに近づくほど落下が速くなる（最大+32%）。
  ランダムで「ラッシュタイム」が発生し、同じ種類が2連続で高速落下（吹き出しが赤い警告表示になる）
- **まんぷくゲージ**: モンスターのすぐ下に「いまの合計＋ターゲットへの近さのバー（目盛り・%付き）」を表示。
  左上のゲージは「しんかゲージ」（3問クリアで進化）
- **モンスターの巨大化**: 満腹度が増えるたびに「ボヨヨン！」と弾みながら少しずつ大きくなり（最大+45%）、
  クリアの瞬間は「ドーン！」とひときわ巨大化。次の問題で初期サイズに戻る（進化時は基本サイズ自体が大きくなる）
- 「0」のひっかけ問題は2問連続で出なかったら必ず次の問題に含まれる。
  0の位に落とすとタワーが揺れて煙＋火花（バチバチッ）が散る
- ブロックは上空 y=40 から落下。高高度では速く（8.8/s）、タワー付近では判定しやすいようゆっくり（1.85/s）になる（`game.js` の `FALL_SPEED_*` で調整可能）
- `window.__game` / `window.__render()` をデバッグ用に公開している
